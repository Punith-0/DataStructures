# 💥⚙️ **DATASTRUCTURES — MASTERING C ONE POINTER BUG AT A TIME**

Welcome to **Punith-0’s DataStructures**, the ultimate C-powered playground where arrays stretch, stacks collapse, and pointers occasionally point to enlightenment (or `NULL`, depending on your luck).

This repo chronicles my journey from basic array acrobatics to full-blown pointer wizardry — all in **pure C**, no training wheels, no abstractions, just raw logic and caffeine-fueled debugging. 💪

---

## 🚀 **OVERVIEW**

This project is a growing library of clean, modular, and well-commented C implementations of essential data structures and algorithms. The goal? To build a rock-solid foundation in DSA using C — one pointer bug at a time 😅.
